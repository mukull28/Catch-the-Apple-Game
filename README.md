🍎 Catch the Apple Game (Python Turtle)

📌 Overview

“Catch the Apple” is a simple interactive game developed using Python and the Turtle graphics module. The player controls a basket to catch falling apples and increase their score.

This project demonstrates basic concepts of computer graphics such as animation, user input handling, and collision detection.

---

🎮 Features

- Smooth basket movement using keyboard controls
- Randomly falling apple
- Real-time score update
- Simple and beginner-friendly design

---

🛠️ Technologies Used

- Python 3
- Turtle Graphics (built-in module)

---

🚀 How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the Python file:

python game.py

---

🎯 Controls

- ⬅️ Left Arrow → Move basket left
- ➡️ Right Arrow → Move basket right

---

📷 Output

A graphical window will open where the player catches falling apples to score points.

---

📚 Learning Outcomes

- Basics of Turtle graphics
- Handling keyboard input
- Game loop implementation
- Collision detection logic
